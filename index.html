⚙️ إعداد Firebase

أدخل بيانات مشروع Firebase. ستجدها في:
Firebase Console → Project Settings → Your apps → SDK setup

1. اذهب إلى console.firebase.google.com
2. أنشئ مشروعاً ← فعّل Authentication → Email/Password
3. فعّل Firestore Database → test mode
4. اضغط أيقونة </> ← سجّل تطبيق ويب ← انسخ الـ config
